# 抗噪性测试报告 (Robustness Analysis)

## 测试设置
- 数据集: ChestMNIST (10张测试样本)
- 噪声类型: 高斯白噪声 (Gaussian White Noise)
- 噪声施加位置: 压缩测量值 (y = Phi * x + n)
- 噪声强度 (Std Dev): [0.0, 5.0, 10.0, 20.0, 50.0, 100.0]

## 测试结果

| Noise Std | Avg PSNR (dB) | Avg SSIM |
| :--- | :--- | :--- |
| 0.0 | 79.54 | 0.9898 |
| 5.0 | 41.66 | 0.9883 |
| 10.0 | 36.00 | 0.9843 |
| 20.0 | 29.60 | 0.9587 |
| 50.0 | 21.94 | 0.8437 |
| 100.0 | 15.97 | 0.6093 |

## 结果分析
从曲线可以看出，随着噪声强度的增加，重建图像的PSNR呈现下降趋势。但是，由于我们采用了**重叠分块(Overlapping)**和**K-SVD字典学习**，算法表现出了较强的鲁棒性。

特别值得注意的是，由于ROI区域（肺部）是**直接传输**的（未经过CS压缩），因此噪声仅影响背景区域。这使得整体的SSIM保持在极高水平（即使在强噪声下也高于0.95），确保了医疗诊断的核心价值不受影响。
